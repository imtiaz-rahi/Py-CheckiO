You have to call monsters from another world in preparation for Halloween.

You are given a string as input value. You have to use this to make monster names and return the maximum number. You can call only 9 kinds of monsters:

  • frankenstein
  • ghost
  • jack
  • mummy
  • skeleton
  • vampire
  • werewolf
  • witch
  • zombie
NOTE:
  • You may make multiple monster names of the same kind.
  • You don't need to use all characters.
  • if you can't make a monster name, returns 0.

Example: 

halloween_monsters('tkjagchso') == 2               # jack, ghost
halloween_monsters('finhtiistchwwaerecnnkt') == 3  # witch, witch, frankenstein

Input: The spell (a string).

Output: A number of monsters (a integer).

Precondition:

  • all(re.fullmatch('[a-z]', i) for i in input)
  • len(input) ≤ 100

How it is used: For calculate materials and products.
